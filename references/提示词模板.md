# 提示词模板

## 通用公式

```
[风格] + [场景@] + [时间戳分镜：景别 + 短句动作 + 台词] + 电影音效，无配乐，无字幕
```

## 核心规则

1. `@「参考图」` → `@`（括号和"参考图"三字删除）
2. 运镜不写，交给模型判断；只标景别
3. 动作用短句，一句话一个动作，禁止长难句
4. 物体描述具体，不能抽象概括，每个镜头独立可描述
5. 统一后缀：`电影音效，无配乐，无字幕`

## 模板1：单人+动作（无台词）

```
{风格}，{场景@}，{角色@}{短句动作描述}。电影音效，无配乐，无字幕
```

示例：
> 古风短剧，翰林院大门前@，宁鸿飞@走出大门，双手抱在脑后，步伐随意。电影音效，无配乐，无字幕

## 模板2：单人+台词

```
{风格}，{场景@}，[景别] {角色@}{短句情绪外化}说"{台词}"，{短句动作}。电影音效，无配乐，无字幕
```

示例：
> 古风短剧，翰林院院内@，[近景] 陈昂@拱手向楚淮安行礼，转身面对宁鸿飞，表情骤变冷厉说"至于宁公子，你身为流民之子，未曾入得宁国公宗谱，恐怕有失体面吧！"抬手指向院中两筐蔬菜和一筐鸭子。电影音效，无配乐，无字幕

## 模板3：多人对话（交替）

```
{风格}，{场景@}，[景别] {角色A@}{反应}说"{台词A}"，{角色B@}{反应}说"{台词B}"，{动作}。电影音效，无配乐，无字幕
```

示例：
> 古风短剧，翰林院院内@，[中景] 楚淮安@骑在马上回头得意说"哟，宁将军，你也来了？"宁鸿飞@被抬着苦笑耸肩说"家父之命，不敢不从啊"。电影音效，无配乐，无字幕

## 模板4：情绪转折（同一角色）

```
{风格}，{场景@}，[景别A] {节拍1短句动作}，[景别B] {节拍2短句动作}。电影音效，无配乐，无字幕
```

示例：
> 古风短剧，翰林院院内@，[中景] 陈茵茵@扫了一眼院中两筐蔬菜和一筐鸭子，嘴角一撇翻了个白眼，视线转向左侧三个打开的宝箱锦缎，眼睛一亮，不由自主往前半步。电影音效，无配乐，无字幕

## 模板5：关键转折/出场

```
{风格}，{场景@}，[景别] {铺垫短句}，{转折/揭晓}，{角色反应}。电影音效，无配乐，无字幕
```

示例：
> 古风短剧，翰林府门外街道，[远景] 桃花树下微风吹落花瓣，一个戴流苏斗笠的女子从远处走来，[近景] 宁鸿飞@怔住嘴唇微张，手从脑后放下垂在身侧，苏锦@轻轻撩开斗笠流苏露出绝世容颜，说"将军，此言当真？"。电影音效，无配乐，无字幕

## 动作描述避坑

| ❌ 长难句 | ✅ 拆成短句 |
|---------|----------|
| 推开椅子站起来走到门前打开门 | 推开椅子 → 站起来 → 走到门前 → 打开门 |
| 转身面向宁鸿飞表情骤变冷厉抬手指着聘礼 | 转身面对宁鸿飞 → 表情骤变冷厉 → 抬手指向两筐蔬菜 |

## 密度检查

写完后检查：
- 总字数 ≤ 300字
- 每3-5秒有一个视觉事件
- 一句话只有一个核心动作
