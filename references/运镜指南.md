# 运镜指南

## 核心原则

**运镜交给视频模型自行判断，提示词中不指定运镜。** 只标景别。

## 景别参考

| 景别 | 作用 | 适用场景 |
|------|------|---------|
| 远景 | 交代环境、展示全貌 | 场景开场、角色出场、留白 |
| 中景 | 展示人物半身+环境 | 对话、日常互动、多人场景 |
| 近景 | 聚焦人物面部+表情 | 情绪反应、关键台词 |
| 特写 | 极度聚焦细节 | 物件特写、情绪高潮、关键动作 |

## 景别与情绪对应

| 情绪/场景 | 推荐景别 |
|----------|---------|
| 对话/日常 | 中景 |
| 情绪转折 | 中景→近景 |
| 关键台词/反应 | 近景 |
| 角色出场 | 远景→中景 |
| 物件/细节强调 | 特写 |
| 留白/沉思 | 远景 |

## 视觉联想规则

前后镜头之间靠景别变化和视觉元素建立联系：
- 上镜头特写某物件 → 下镜头远景中出现该物件（环境锚定）
- 上镜头角色视线看向某方向 → 下镜头切到该方向的焦点（视线引导）
- 上镜头远景交代空间 → 下镜头中景聚焦人物（空间→人物递进）

## 为什么不写运镜

- 短剧运镜不需要复杂，简单直接最好
- 过于精确的运镜约束反而限制模型发挥，效果不好
- 动作描述越详细，模型越能自行判断合适的运镜
